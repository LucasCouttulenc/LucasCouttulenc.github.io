Black Jack Simulator

En Desarrollo

La imagen es sólo ilustrativa

  • Java

  • JavaFx

Un juego de Black Jack que te permite tanto jugar manualmente como sacar estadísticas personalizadas. Dichas estadísticas pueden sacarse tanto con la estrategia básica como con el sistema de conteo de cartas HiLo.

Estoy haciendo este proyecto siguiendo el paradigma de Programación Orientada a Objetos (POO).
Además, estoy respetando el patrón MVC (Model-View-Controller) para separar la lógica de la vista. De esta manera, puedo crear dos vistas diferentes, la CLI (Command Line Interface) y la GUI (Graphical User Interface).
La CLI ya está mayormente finalizada, mientras que la GUI está aún en desarrollo.
Por lo tanto, aún no tengo un ejecutable ni un video demostrativo de la aplicación. Pero estará disponible tan pronto como esté la interfaz gráfica hecha.
